# AI Resume Scanner 🚀

An AI-powered Resume Scanner built using Python and Flask that helps companies and institutions automate the recruitment process by analyzing resumes, extracting important keywords, comparing candidate skills with company requirements, and automating communication.

---

## 📌 Features

- 📄 Resume Upload & PDF Processing
- 🔍 Keyword Extraction from Resumes
- 🤖 AI-based Resume Analysis
- 💬 Integrated Chatbot Support
- 📧 Automated Email Notifications
- 🏢 Company Requirement Matching
- ⚡ Lightweight and Fast Processing

---

## 🛠️ Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite3
- PDFMiner
- ChatterBot
